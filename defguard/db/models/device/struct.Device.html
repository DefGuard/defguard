§id: I§name: String§wireguard_pubkey: String§user_id: Id§created: NaiveDateTime§device_type: DeviceType§description: Option<String>§configured: bool

Whether the device should be considered as setup and ready to use or does it require some additional steps to be taken. Not configured devices won’t be sent to the gateway. It is assumed that an unconfigured device is already added to all networks it should be in, but it’s not ready to be used yet due to e.g. public key not properly set up yet.
