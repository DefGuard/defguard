AclAlias
Database representation of an ACL alias. Aliases can be used to define the destination part of an ACL rule so that it’s easier to create new rules with common restrictions. In addition to the AclAlias we provide AclAliasInfo and ApiAclAlias that combine all related objects for easier downstream processing.

PortRange
Representation of port range. Those are stored in the db as [PgRange<i32>]. Single ports are represented as single-element ranges, e.g. port 80 = Range(80, 81) since upper bound is excluded by convention.
